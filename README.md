# Bloomily Social Media Backgrounds

On-brand background images for Bloomily social media content.

## Brand Colors
- **Gold:** #F5B72D
- **Amber:** #E5A520
- **Honey:** #FFCF5C
- **Warm cream:** #FFF8E6
- **Subtle bg:** #FFFBF2

## Formats

### `/square` (1080x1080)
Instagram feed posts, Facebook posts

### `/portrait` (1080x1350)
Instagram feed (4:5 aspect ratio)

### `/story` (1080x1920)
Instagram Stories, Reels, TikTok

### Root folder (4K originals)
High-res source files (5632x3072)

## Styles
- **wave** - Soft organic flowing shapes
- **corners** - Gradient blobs in corners
- **circles** - Subtle overlapping circles
- **gradient** - Smooth vertical gradient
- **diagonal** - Diagonal gradient sweep
